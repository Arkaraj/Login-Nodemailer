## To run on machine
git clone
```npm install```
Dump Sql code in Database

Make .env file with attributes 

1. HOST 
2. USER_DB 
3. PASSWORD
4. DB_NAME
5. MAIL_USER
6. MAIL_PASS
7. USR

If **nodemon** installed: ```npm start``` 

or

```node app.js```
